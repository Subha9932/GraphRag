Core Primitives

Modular design

The system is built on three specialized pillars that enable it to act as a senior engineer.

Intent — SEE

Enables the AI to UNDERSTAND

Classifies the user's request to determine the best strategy.

  • Router: Decides which agents to activate.
  • Ambiguity Detection: Asks clarifying questions.
  • Context Aware: Remembers conversation history.
GraphRAG — THINK

Enables the AI to KNOW

Retrieves deep, connected knowledge from the codebase.

  • Prompt Engineering: Custom prompts (extract_graph.txt) specifically tuned for identifying code entities.
  • Standard CLI: Uses Microsoft's robust GraphRAG implementation.
  • Hybrid Search: Combines vector search with graph traversal.
Orchestrator — ACT

Enables the AI to COORDINATE

Manages flow, state, and final output generation.

  • Parallelism: Runs agents concurrently.
  • State Reducers: Merges data from multiple sources.
  • Synthesis: Compiles disjoint facts into a narrative.

Components
